# Publishing Your Biology 30 Review Hub to GitHub Pages

## What you'll need
- A free GitHub account (github.com)
- GitHub Desktop app (free — desktop.github.com)
- About 20 minutes

---

## STEP 1 — Create a free GitHub account

1. Go to **https://github.com** and click **Sign up**
2. Choose a username (e.g. `dingram-bio30` or `ursa-biology`)
3. Verify your email address
4. Choose the **Free** plan

---

## STEP 2 — Download GitHub Desktop

1. Go to **https://desktop.github.com**
2. Download and install GitHub Desktop
3. Sign in with your GitHub account when prompted

---

## STEP 3 — Create a new repository

1. In GitHub Desktop, click **File → New Repository**
2. Fill in the form:
   - **Name:** `bio30-review` (no spaces — this becomes part of your URL)
   - **Description:** Biology 30 Diploma Exam Review Hub
   - **Local path:** Pick any convenient folder (e.g. your Desktop)
   - ✅ Check **Initialize this repository with a README**
3. Click **Create Repository**

---

## STEP 4 — Copy your files into the repository folder

1. Open File Explorer and navigate to the repository folder GitHub Desktop just created
   (it will be something like `C:\Users\YourName\Desktop\bio30-review`)
2. Copy **all contents** of your `Biology 30 Comprehensive Review` folder into this repo folder:
   - `index.html`
   - `Bio30_Diploma_Review.html`
   - The entire `resources` folder (with all subfolders and PDFs)
3. Your repo folder should now look like:
   ```
   bio30-review/
     index.html
     Bio30_Diploma_Review.html
     resources/
       unit_a/
       unit_b/
       unit_c/
       unit_d/
       diploma/
   ```

---

## STEP 5 — Commit and upload (push) to GitHub

1. Switch back to GitHub Desktop
2. You will see a long list of changed files — that's all your new files
3. In the bottom-left, type a **Summary** (e.g. `Initial upload of Bio 30 Review Hub`)
4. Click **Commit to main**
5. Click **Publish repository** (top of the screen)
   - Make sure **Keep this code private** is **unchecked** (must be public for free GitHub Pages)
6. Click **Publish Repository**
7. Wait for the upload to complete — the 508MB of PDFs may take 5–15 minutes depending on your connection

---

## STEP 6 — Enable GitHub Pages

1. Go to **https://github.com** and click on your new `bio30-review` repository
2. Click the **Settings** tab (gear icon)
3. In the left sidebar, click **Pages**
4. Under **Branch**, select **main** from the dropdown, leave the folder as `/ (root)`
5. Click **Save**
6. GitHub will show a message: *"Your site is ready to be published"*
7. After 2–3 minutes, refresh the page — you'll see:
   **"Your site is published at https://YOUR-USERNAME.github.io/bio30-review/"**

---

## STEP 7 — Add to your Canvas course

1. In Canvas, go to your **Biology 30 course → Modules**
2. Click **+** to add a new item to a module
3. Choose **External URL**
4. Paste your GitHub Pages URL: `https://YOUR-USERNAME.github.io/bio30-review/`
5. Give it a title (e.g. *Biology 30 Diploma Review Hub*)
6. Check **Load in a new tab** ← recommended so it opens full-screen
7. Click **Add Item**

That's it! Students click the link in Canvas and land on your full interactive hub.

---

## Your permanent URL will be:
```
https://YOUR-GITHUB-USERNAME.github.io/bio30-review/
```

---

## When you want to update the hub in the future

1. Make any changes to files in the `bio30-review` folder on your computer
2. Open GitHub Desktop → it will detect the changes
3. Write a summary, click **Commit to main**, then **Push origin**
4. The live site updates automatically within 1–2 minutes

---

## Notes

- **PDFs are publicly accessible** via the URL — anyone with the link can view them.
  This is fine for course materials; just be aware if any documents are proprietary.
- Your GitHub Pages site is **free and permanent** as long as your account is active.
- If you ever want to make the repo private, you'll need a paid GitHub plan for Pages.
